# ТЗ: Frontend

## 1. Общее описание
Визуальный редактор блок-схем на базе React + React Flow.

## 2. Технологический стек
- React 18+
- React Flow (MIT)
- TypeScript
- Axios / Fetch API
- CSS Modules / Styled Components (TBD)

## 3. Основные компоненты

### 3.1 Визуальный редактор схем
- Canvas для размещения нод
- Панель инструментов с типами нод
- Drag & drop интерфейс
- Соединение нод (edges)

### 3.2 Панель настройки ноды
- Форма с параметрами выбранной ноды
- Валидация полей
- Предпросмотр JSON конфигурации

### 3.3 Список схем
- CRUD операции
- Фильтрация/поиск
- Статус схемы (draft/active/archived)

### 3.4 Панель выполнения
- Запуск схемы (Manual)
- Отображение текущего статуса
- История выполнений
- Pause/Resume/Stop controls
- Логи выполнения

### 3.5 Настройки Webhook
- Генерация URL для webhook
- Отображение примеров payload
